# Employee-Directory

##Description

This assignment utilization of react to launch an employee directory application. The application gets random employees from an API via Axios and user is able to both search in the search bar for an employee or filter based on one of the predefined columns.

This assignmenet entails React which is a new based programming software and it allows developers to make and launch apps fairly quickly due to its lightweight architecture.
